# Arduino Reaction Speed Game 







A simple Arduino-based LED project that controls one or more LEDs using basic programming logic.







## Features



- Reaction-based game using physical buttons



- Randomized delays to prevent prediction



- Multi-stage LED + buzzer signal sequence







## Technologies Used



- Arduino (C/C++)

- LEDs (Green, Yellow, Red, White x2)

- Push Buttons (x2)

- Buzzer

- Resistors

- Breadboard & Jumper Wires 







## What I Practiced



- Digital input/output control 



- Delay Logic



- Circuit building 



- Debugging hardware connections



- Button handling with INPUT_PULLUP 

- Random timing







## Future Improvements



- Add reaction time scoring system

- Display results using LCD or 7-segment display
